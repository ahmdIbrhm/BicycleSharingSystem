Bicycle Sharing System

Get Started Try it out
This website is done for sharing bicycles in different cities. This website allows the user to add different cities in order to target them.
In the tab "Add Cities", we can find some values that the user should add in order to add a specific city. Below, we can find the explanation of each input:
  • City name: The name of the city that you want to add (Lyon, Paris, Marseille, ...)
  • Dynamic data link: The uri of the data that corresponds to the city.
  • Link's file type: The type of the file in the link before (JSON, XML, CSV).
  • Path of station id: The path of the key that defines each station in the city separated by a dot '.'.
  • Path of array of stations: The name of the array of stations (Iterator).
